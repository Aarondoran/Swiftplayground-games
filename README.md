# Swiftplayground-games

My collection of swift apps.

## Quick Instructions
1. Download the playground file or folder from this repository (click the file or folder, then Download).
2. Download the Swift Playgrounds app if you don't already have it.
3. Unzip/uncompress the file to get the .swiftpm file
4. Add the downloaded file to your Swift Playgrounds iCloud Drive folder:
   - On iPad: Files app → iCloud Drive → Playgrounds (or create a Playgrounds folder) → move the playground folder there.
5. Open Swift Playgrounds and the app should appear; select the playground to open and run it.
