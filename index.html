Swiftplayground-games

My collection of swift apps.

Quick Instructions

  1. Download the playground file or folder from this repository.
  2. Download the Swift Playgrounds app if you don't already have it.
  3. Unzip the file to get the .swiftpm file.
  4. Add the downloaded file to your Swift Playgrounds iCloud Drive folder.
  5. Open Swift Playgrounds and the app should appear.
